How to start up with server

Create virtual env with the following command:
python3 -m venv venv/

Then activate it:
source venv/bin/activate

Load important packages:
pip install flask
pip install sys
pip install tensorflow
pip install segmen
pip install pickle
pip install pretrainedmodels
pip install torch
pip install torchvision
pip install typing
pip install PIL
pip install requests
pip install io
pip install matplotlib

Run server:
python server.py
